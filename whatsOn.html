| whatsOn |

whatsOn is a user-friendly way for movie lovers to spend more time watching, rather than searching.
By utilizing the Guidebox and Gracenote APIs we are able to deliver up-to-date data to the user.

At any time, the user is able to select what streaming services they want to search. This will not only improve search results. It will also generate recommended movies based upon the selected services.

If the user desires to hit a theater, they are able to search most local theaters for current titles and showtimes.
