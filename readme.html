| README Generator |

The README Generator allows a developer to create professionally styled README files from their favorite text editor.
It includes functionality to add a header style text as well as normal paragraph text.
At the end, a developer will be able to enter their GitHub username; where the application will hit the GitHub API and pull their GitHub e-mail and profile picture.
