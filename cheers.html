| cheers |

Cheers is a fast-paced, real-time, online multiplayer game that can be
played by 2 or more people. Cheers utilizes socket.io to handle multiple client connections.

Players are given different either a true/false or an open-ended prompt that asks for user input.
Each time a player answers true or false question incorrectly, the game will increase the amount of "vision blur" on that player's screen.
Getting one correct will reset this effect.
An open-ended prompt will ask for input from each player, and then allow the group to vote on which one they like the best.
The choice with the highest amount of votes is displayed along with the winner's name.
